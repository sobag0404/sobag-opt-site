# Sobag Opt Site

Тестовый прототип B2B-сайта Sobag для оптовых продаж текстильной продукции с принтами.

## Что внутри

- каталог с тестовыми товарами;
- поиск, категории и фильтры;
- заявка-корзина для оптового заказа;
- расчет скидки по сумме;
- форма заявки и бриф на производство своего принта.

Проект статический: достаточно открыть `index.html` или задеплоить папку на Vercel.
