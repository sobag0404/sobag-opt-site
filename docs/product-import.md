# Массовая загрузка товаров и фото

## Логика

Одна строка в шаблоне = один основной артикул / один принт. Сайт сам создает варианты по столбцам:

- `Типы товара`: например `Подушка, Наволочка`
- `Размеры`: например `30x30, 35x35, 40x40, 45x45, 50x50`
- `Материалы`: например `Велюр, Габардин`

Фото ищутся по папке основного артикула. Пример:

```text
Фото товаров/
  10345/
    1.jpg
    2.jpg
    3.jpg
  10346/
    1.png
    2.webp
```

`1.jpg` обычно становится главным фото, остальные уходят в галерею. Лучше держать фото квадратными `1200x1200 px`, JPG/WebP.

## Если список артикулов уже есть

Скачать шаблон можно из админки кнопкой `XLSX-шаблон` или из файла:

```text
templates/sobag-products-template.xlsx
```

После заполнения можно загрузить таблицу через админку или прогнать локальный импортер.

## Если списка артикулов нет

Импортер умеет собрать черновой шаблон из названий папок с фото:

```powershell
python tools/product_importer.py scan-photos --photos "C:\Path\YandexDisk\Фото товаров" --out local-import-output\products-from-photo-folders.xlsx
```

После этого файл `products-from-photo-folders.xlsx` можно открыть, дописать названия, категории, подборки, праздники, цены и загрузить обратно.

## Локальный импорт

```powershell
python tools/product_importer.py import --table local-import-output\products-from-photo-folders.xlsx --photos "C:\Path\YandexDisk\Фото товаров"
```

Результат:

- `assets/imported-products/` - скопированные фото;
- `data/products.import.json` - подготовленные товары для будущего подключения к сайту/backend;
- `data/import-report.csv` - отчет по строкам и фото.

Эти выходные папки сейчас добавлены в `.gitignore`, чтобы случайно не загрузить тысячи фото в GitHub.

## Яндекс Диск

Самый удобный вариант: установить приложение Яндекс Диска и синхронизировать папку с фото на компьютер. Тогда для импортера это обычная локальная папка, например:

```powershell
--photos "C:\Users\SoBag\YandexDisk\Фото товаров"
```

Давать пароли или токены в чат не нужно. Если позже понадобится импорт напрямую по ссылкам/API, лучше подключать это отдельным безопасным способом.
